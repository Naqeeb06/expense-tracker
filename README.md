# Expense Tracker (Python CLI)

A command-line Expense Tracker built with Python that helps users manage their daily expenses. The application stores data in a CSV file, allowing expenses to persist between program runs.

## Features

- Add new expenses
- View all expenses
- Edit existing expenses
- Delete expenses
- Calculate total expenses
- Filter expenses by category
- Input validation for user inputs
- CSV-based persistent storage

## Technologies Used

- Python 3
- CSV Module
- OS Module

## Project Structure

```
expense-tracker/
│── main.py
│── expenses.csv
└── README.md
```

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/your-username/expense-tracker.git
```

2. Navigate to the project directory:

```bash
cd expense-tracker
```

3. Run the application:

```bash
python main.py
```
